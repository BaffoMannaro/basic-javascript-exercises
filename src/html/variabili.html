Using variables

javascript.js
var num1; var num2; num1 = 3; num2 = 5; console.log('num1 = ' + num1); console.log('num2 = ' + num2); var num3; num3 = num1; num1 = num2; num2 = num3; console.log('num1 = ' + num1); console.log('num2 = ' + num2);
"This exercise swaps the values of two variables (`num1` and `num2`) using a temporary variable (`num3`).
Initially, `num1` is 3 and `num2` is 5; after the swap, `num1` becomes 5 and `num2` becomes 3."
